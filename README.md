# Brahim Academy

![Website Preview](images/Screenshot%202026-05-06%20125932.png)

## Description

Brahim Academy is an online learning platform dedicated to providing high-quality courses in web development and programming. Our platform offers a comprehensive curriculum designed to help learners build practical skills and advance their careers in tech.

## Features

- **Responsive Design**: Built with Bootstrap 5 for optimal viewing on all devices
- **Course Catalog**: Showcase of popular courses including HTML & CSS, JavaScript, React, and more
- **Pricing Plans**: Flexible subscription options including monthly and lifetime plans
- **Interactive Elements**: Carousel for course browsing, modals for enrollment, and accordion for FAQs
- **Contact Information**: Easy access to location, phone, and email details
- **Newsletter Signup**: Stay updated with the latest news and updates

## Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Custom styling and responsive design
- **Bootstrap 5.3.3**: Front-end framework for components and utilities
- **JavaScript**: Bootstrap's JavaScript for interactive components
- **Bootstrap Icons**: Icon library for visual elements

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/brahim1-dev/baro-academy.git
   ```

2. Navigate to the project directory:
   ```bash
   cd baro-academy
   ```

3. Open `index.html` in your preferred web browser.

## Usage

Simply open the `index.html` file in any modern web browser to explore the Baro Academy website. The site is fully responsive and works seamlessly across desktop, tablet, and mobile devices.
